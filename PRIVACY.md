# Privacy Policy – Azure Naming Helper

Azure Naming Helper verarbeitet, speichert oder überträgt **keine personenbezogenen Daten**.

## Datenspeicherung
Die Erweiterung speichert ausschließlich folgende Konfigurationswerte lokal über `chrome.storage.sync`:

- Environment (Env)
- Location
- Application
- Ressourcentyp-Auswahl
- Naming-Patterns pro Ressourcentyp

Diese Daten werden **nicht** mit externen Diensten geteilt oder übertragen.

## Keine externe Kommunikation
Das Plugin führt **keine Netzwerkrequests** aus und sendet **keine Telemetrie**.

## Keine Analytics, kein Tracking
Es gibt keinerlei Nutzungsanalyse oder versteckte Datensammlung.

## Offene Nutzung
Das Tool kann sicher innerhalb Unternehmensnetzwerke genutzt werden.  
Alle Konfigurationen bleiben vollständig lokal im Browser gespeichert.
