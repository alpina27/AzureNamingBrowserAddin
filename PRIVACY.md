# Privacy Policy – Azure Naming Helper
## EN-US
Azure Naming Helper processes, stores or transmits **no personal data**.

## Data storage
The extension only stores the following configuration values locally via `chrome.storage.sync`:
- Environment (Env)
- Location
- Application
- Resource type-selection
- Naming-Patterns per resource type

This data are **not** shared or transmitted with external services.

## No external communication
The plugin executes **no network requests** and sends **no telemetry**.

## No analytics, no tracking
There is no usage analysis or hidden data collection.

## Open use
The tool can be used securely within corporate networks.
All configurations remain fully stored locally in the browser.


## DE-DE
Azure Naming Helper verarbeitet, speichert oder überträgt **keine personenbezogenen Daten**.

## Datenspeicherung
Die Erweiterung speichert ausschließlich folgende Konfigurationswerte lokal über `chrome.storage.sync`:

- Environment (Env)
- Location
- Application
- Ressourcentyp-Auswahl
- Naming-Patterns pro Ressourcentyp

Diese Daten werden **nicht** mit externen Diensten geteilt oder übertragen.

## Keine externe Kommunikation
Das Plugin führt **keine Netzwerkrequests** aus und sendet **keine Telemetrie**.

## Keine Analytics, kein Tracking
Es gibt keinerlei Nutzungsanalyse oder versteckte Datensammlung.

## Offene Nutzung
Das Tool kann sicher innerhalb Unternehmensnetzwerke genutzt werden.  
Alle Konfigurationen bleiben vollständig lokal im Browser gespeichert.
